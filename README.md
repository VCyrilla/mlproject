# ML-Driven File Execution Mapping for Malware Detection

Bachelor of Science in Informatics and Computer Science Project.

Supervisor: Dr. Stephen Obonyo

## Overview
This repo contains code for detecting polymorphic malware using ML (unsupervised clustering, LSTM sequences, zero-shot learning).

## Setup
- Create venv: python3 -m venv malware_env
- Activate: source malware_env/bin/activate
- Install deps: pip install numpy pandas scikit-learn torch lief requests git+https://github.com/elastic/ember.git
